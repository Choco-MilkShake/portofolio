# Daniel Abner | Portfolio

Website portofolio yang dibangun menggunakan Next.js (App Router), React, dan GSAP untuk animasi.

## Fitur
- Desain antarmuka modern dengan gaya "dark mode"
- Animasi GSAP (ScrollTrigger & Intro Animasi)
- Efek latar belakang bintang interaktif
- *Responsive Layout*

## Pengembangan (Development)

Untuk menjalankan server pengembangan lokal:

```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
# atau
bun dev
```

Buka [http://localhost:3000](http://localhost:3000) di peramban Anda untuk melihat hasilnya.

## Teknologi Utama
- **Next.js 14+** (Framework React)
- **React 18+** (Library UI)
- **GSAP** (GreenSock Animation Platform)
- **Vanilla CSS** (Penggayaan tanpa framework)
